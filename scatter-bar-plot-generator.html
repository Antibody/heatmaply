Make Scatter Plots Combined with Bar Plots Online with Scatter Plot Bar Web App

Free web app alternative to Excel and GraphPad Prism to make dot plots combined with bar plots

Display significance levels with brackets just as in GraphPad Prism on scatter plots combined with columns or barplots

The scientific data presentation often requires visualisation of the individual datapoints. This web app does that. You can now easily create dot plots overlaying columns, violin, box, and bar plots online. Or just add a crossbar (representing means) to your scattered datapoints. That one is at the bottom of the "Additonal Plot Options" section but you need to build e.g. columns or violin plots first. Crossbar replaces black and white columns but overlays box and violin plots.

Data entry; either manually enter the data into the table in one tab of the app (it is a default one) or uppoad a comma separated file (CSV) from the second tab (marked in red font). From that tab you can also upload a table for making grouped bar plots combined with scatter plots. However, if you need no individual datapoints on you graphs you can remove them by reducing their opacity to zero.

The statistics section, the one below color and datapoint controls panel, is an online calculator for Mann-Whitney U Test (Wilcoxon), Welch t-test, paired t-test. For multiple comparisons you can run, depending on your data, ANOVA followed by Tukey HSD (honestly significant difference), and/or Kruskal-Wallis with Dunn's tests. I recently added a possibility to display significance levels for all statistical tests on the plot. That is, you can now click on the tickbox 'Choose display style and font size' to display a p value on the bar plot, accompanied by brackets indicating compared columns

One can generate online a line graph connecting the means and combined with error bars. The data needs to be in columns and the headers must be numeric. In addition, only one graph per plot is possible, currently.

The app is based on R, ggplot2, and shiny. It is hosted by shinyapps.io for free
